# LVScript #
____________________

## Что это такое? ##
LiveStudioScript - язык программирования(скриптовый язык, в оболочке библиотеки Python'а), который позволит вам:

    •Базовые команды - print(put), input(put_on) и т.д.

    •Математические действия
    
    •Этичный хакинг - информация о IP, номере телефона и многом другом.

    •Парсинг сайтов - куки, код сайта, содержимое страницы и т.д.

    •Работа с GIT(GitHub - публикация кода, обновление, и многое другое...)

    •Доступ к нейросетям последней версии(3.5 Turbo и 4.0)

    •Коннвертация кода в EXE, DEB, DMG пакеты

    •Создание своих библиотек

    •Создание локальных и глобальных серверов(в том числе и подключение к ним)

    •И многое другое ;)
## Как скачать? ##
Для скачивания языка:

    Linux:

         sudo apt update && sudo apt upgrade && sudo apt install python3 && sudo apt install python3-pip && sudo apt install git && sudo pip install lvscriptlvst
    
    MacOS:

        Тоже самое что и Linux =)

    Windows:

        Скачайте python3 с официального сайта(https://www.python.org/downloads/) последней версии, зайдите в PowerShell, pip install lvscriptlvst, и дальше смотрите пункты для написания кода :)
